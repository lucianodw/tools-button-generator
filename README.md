# Button Generator

A simple button generator used as a helpful tool for the team at Direct Wines.  It outputs the the actual button & the classnames that should be used for that button.

### Stack
- AngularJS
- SASS / Bootstrap
- Grunt
- NPM / Bower


### Run Locally
```sh
npm install
grunt serve
```
